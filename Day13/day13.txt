0) Feature Selection:-
    From the above data visual, we can decide which algo will work Here,Linear regression will work because Dataset is easily separable into two parts.

1) Extract Input and Output cols:-
    X = df.iloc[:,0:2] -> get index cgpa and iq
    y = df.iloc[:,-1] -> get last column placemnet

3) Train Test Split:-
   from sklearn.model_selection import train_test_split
   X_train, X_test, y_train, y_test = train_test_split(X,y,test_size=0.1)

   0.1 -> 10% data for testing and 90% data for training
   i.e out of 100 rows, 90 rows for training and 10 rows for testing

2) For Scaling the value:-
   from sklearn.preprocessing import StandardScaler
   scaler = StandardScaler()
   X_train = scaler.fit_transform(X_train)
   X_test = scaler.transform(X_test)

4) Train the model:-
    We have decided to use Logistic Regression classifier because output is 0 and 1

    from sklearn.linear_model import LogisticRegression
    clf = LogisticRegression()
    clf.fit(X_train,y_train)

5) Evaluate the model:- working on test data
    y_pred = clf.predict(X_test)

    for accuracy calculation:-
    from sklearn.metrics import accuracy_score
    accuracy = accuracy_score(y_test,y_pred)

    O/P 0.9 or 90% i.e 10 input diya toh ye aapko 10 main se 9 sahi predict karke de raha hai aur ek galat predict kar raha hai.
